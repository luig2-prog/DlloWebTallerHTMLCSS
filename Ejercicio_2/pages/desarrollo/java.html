Lo que aprenderás
Crear aplicaciones Web y Empresariales Java con competencias reales para conseguir oportunidades en el mercado laboral y profesional como programador Java Dominar los fundamentos de Java, conexión a base de datos, hasta la creación de aplicaciones Web con Servlets y JPS's. A este nivel se le conoce como Java Jr. Dominar las tecnologías Java Empresariales (Java EE o antes Java2EE) como EJB's, JPA, Web Services, JSF, PrimeFaces. A este nivel se le conoce como Java Sr. Al finalizar dominarán todos los temas anteriores, ya que cada nivel se apoya del anterior para obtener los niveles que el mercado laboral Java necesita. Sienta las bases para comenzar tu preparación de certificación como programador Java
